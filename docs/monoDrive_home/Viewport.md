# Viewport

The Viewport sensor allows a client to configure the camera that is displayed in 
the monoDrive Simulator or Scenario Editor. 

## Configuration

``` json
{
  "type": "ViewportCamera",
  "fov": 60.0,
  "location": {
      "x": 0.0,
      "y": 0.0,
      "z": 0.0
  },
  "rotation": {
      "pitch": 0.0,
      "roll": 0.0,
      "yaw": 0.0
  }
}
```

**fov:** Sets the horizontal angle of view of the viewport camera.
